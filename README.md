# Portafolio Personal

Bienvenido al repositorio de mi portafolio personal desarrollado con **Astro**, **React**, **Tailwind CSS** y estilizado con la paleta **Gruvbox Dark**.

## Tecnologías Utilizadas

- **Framework:** [Astro](https://astro.build/)
- **UI:** React & Framer Motion
- **Estilos:** Tailwind CSS (Theme: Gruvbox Dark)
- **Despliegue:** GitHub Pages

## Instalación y Ejecución Local

1. Clona el repositorio:
   ```bash
   git clone https://github.com/victoraleja1/my-portafolio.git
   ```
2. Instalar Dependecias:
   ```bash
   npm install
   ```
3. Iniciar el Servidor de Desarrollo:
   ```bash
   npm run dev
   ```
